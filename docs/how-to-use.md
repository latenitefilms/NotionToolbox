# How To Use

## Usage

### Merger
1. Open the app and select the **Merger** tab
2. Drop or select your Local Spreadsheet CSV file
3. Drop or select your Notion Database CSV export
4. Click **Merge CSVs** to configure column mapping
5. Adjust mappings as needed and confirm
6. Save the merged CSV file

---

### Importer
1. Select the **Importer** tab
2. Click **Settings** to configure your Notion connection:
   - Enter your Notion Integration Token
   - Enter your Database ID
   - Test the connection
3. Either:
   - Use **Use Merged Data** button (if you have merged data from the Merger tab)
   - Or drop/select a CSV file to import
4. Configure column mappings between CSV columns and Notion properties
5. Click **Update Notion** to import the data

---

## Setting up Notion Integration

1. Go to [Notion Integrations](https://www.notion.so/my-integrations)
2. Click **New integration**
3. Give it a name and select the workspace
4. Copy the **Internal Integration Token**
5. Share your database with the integration:
   - Open your Notion database
   - Click **...** → **Add connections** → Select your integration
